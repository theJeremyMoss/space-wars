## How to Play

### Controls
- **Arrow Keys (Left/Right)**: Move your ship horizontally
- **Spacebar**: Fire bullets at enemies

### Gameplay
- Destroy enemies in the formation above to earn points
  - Top two rows (TIE Interceptors): Worth 100 points each
  - Bottom two rows (TIE Fighters): Worth 50 points each
- Only enemies in the bottom row can fire back
- You have 3 lives
